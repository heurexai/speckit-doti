# Constitution Template

Use this template to initialize or revise `.doti/memory/constitution.md`.

Required principles:

- deterministic commands own gates;
- advisory checks are not proof;
- the template engine owns static generation;
- the scaffold CLI owns finishing and verification;
- public hygiene is mandatory;
- generated repos must not depend on shell runners;
- engineering discipline: root-cause failures (never patch symptoms), validate every assumption by reading/running/observing (prove claims, never assume), take no shortcuts, hold a 95%-confidence bar, surface genuine blockers instead of guessing, and report honestly;
- operator decisions: surface any blocking question with full context, each option's pros/cons/consequence, a recommendation with reasoning, stated assumptions, and a confidence level.
