# Support

Use the project issue tracker for public support:

- bug reports: use the bug report issue form;
- feature requests: use the feature request issue form;
- contribution questions: start with `CONTRIBUTING.md` and open a focused discussion or issue when the answer is not covered there.

Do not post secrets, private repository content, customer data, or vulnerability details in public issues. For security-sensitive reports, follow `SECURITY.md`.

This project is maintained as open source. Response times are best-effort.
